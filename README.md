## 🔬 Featured Work

<div align="center">

> **Plaksha Streams Pipeline** — Production Video Analytics Platform
>
> `Kafka` → `PySpark Structured Streaming` → `ML Inference` → `PostgreSQL` → `Real-time Dashboard`

| | |
|:---|:---|
